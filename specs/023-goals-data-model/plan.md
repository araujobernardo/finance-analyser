## Implementation

### Migration: `src/db/migrations/0005_goals_schema_complete.sql`

```sql
ALTER TABLE goals
  ADD COLUMN category_name  varchar(100),
  ADD COLUMN current_amount numeric(15, 2),
  ADD COLUMN updated_at     timestamptz NOT NULL DEFAULT now();
```

### Drizzle Schema: `src/db/schema.ts`

Add three fields to the `goals` table definition:

```ts
categoryName: varchar("category_name", { length: 100 }),
currentAmount: numeric("current_amount", { precision: 15, scale: 2 }),
updatedAt: timestamp("updated_at", { withTimezone: true })
  .defaultNow()
  .notNull(),
```

`Goal` and `NewGoal` types (already exported via `$inferSelect`/`$inferInsert`) gain the new fields automatically — no manual type changes needed.

### Column semantics

| Column           | Nullable | Default | Populated by                                                |
| ---------------- | -------- | ------- | ----------------------------------------------------------- |

| `current_amount` | YES      | null    | FA-GOAL-003 (progress calculation)                          |
| `updated_at`     | NO       | `now()` | DB DEFAULT on insert; FA-GOAL-002 PATCH sets it via Drizzle |
