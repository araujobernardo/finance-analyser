# Implementation Plan: Backend API Server Deployment

**Branch**: `009-api-server-deploy` | **Date**: 2026-05-06 | **Spec**: [spec.md](spec.md)

---

## Summary

Deploy a minimal Express server to Railway that passes a health check and provides a structured foundation for future REST API routes. No auth, no database queries, no data endpoints. The deliverable is a running Railway service that returns `{ "status": "ok", "timestamp": "<ISO>" }` from `GET /health`, enforces CORS via `CORS_ORIGIN`, returns JSON errors universally, and can be started locally with `npm run server:dev`.

---

**Scale/Scope**: Single Railway service; no auth or data routes in this feature; ~5 source files

### Task 5 — `src/server/middleware/errorHandler.ts`

Catches all errors forwarded via `next(err)` or thrown synchronously in Express 5 route handlers.

```typescript
import type { ErrorRequestHandler } from "express";

interface AppError extends Error {
  status?: number;
}

export const errorHandler: ErrorRequestHandler = (
  err: AppError,
  _req,
  res,
  _next,
) => {
  const status = err.status ?? 500;
  res
    .status(status)
    .json({ error: err.message || "Internal Server Error", status });
};
```

A 404 handler is also registered in `index.ts` (before the error handler) to catch unknown routes:

```typescript
app.use((_req, res) => {
  res.status(404).json({ error: "Not Found", status: 404 });
});
```

### Task 6 — `src/server/index.ts`

```typescript
import express from "express";
import cors from "cors";
import healthRouter from "./routes/health.ts";
import { errorHandler } from "./middleware/errorHandler.ts";

const app = express();
const PORT = process.env.PORT ? Number(process.env.PORT) : 3001;
const CORS_ORIGIN = process.env.CORS_ORIGIN;

if (!CORS_ORIGIN) {
  console.warn(
    "[server] CORS_ORIGIN is not set — all cross-origin requests will be denied",
  );
}

app.use(
  cors({
    origin: CORS_ORIGIN ?? false,
    credentials: true,
  }),
);
app.use(express.json());

app.use(healthRouter);

// 404 — must come after all routes
app.use((_req, res) => {
  res.status(404).json({ error: "Not Found", status: 404 });
});

// Global error handler — must come last, after the 404 handler
app.use(errorHandler);

app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});
```
