# Finance Analyser — Product Context

**Product type:** Tool (design serves the product, not the brand)
**Primary user:** A single regular person (the developer) managing personal finances from NZ bank CSV exports
**Primary use case:** Upload bank statements, get automatic categorisation, visualise spending patterns, ask the AI assistant questions about money
**Design register:** Product UI — clarity and insight over visual flair
**Tone:** Calm, trustworthy, data-forward. Never alarming, never playful.
**Reference app:** Monarch Money (monarchmoney.com)
**Design system:** See docs/design-system.md for all tokens and patterns
**Anti-patterns:** See .claude/skills/impeccable/ for full banned list

## Key constraints

- Dark mode only
- NZD currency throughout
- Mobile-friendly but desktop-primary
- No onboarding flow — user uploads CSV and sees data immediately
